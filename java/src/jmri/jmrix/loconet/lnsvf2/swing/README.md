The jmrix.loconet.lnsvf2.swing Java package defines LocoNet-specific functionality 
supporting GUI operations related to LocoNet "SV Format 2" devices.

LocoNet "SV Format 2" messaging is defined in the "LocoNet Personal Use Edition 
1.0 Extension" document which covers SV Message Programming Formats.  This document
is available via the "LocoNet-Hackers" group on groups.io.  General support for
SV Format 2 LocoNet messaging is via that groups.io group.

Some of the message formats used in this package are Copyright Digitrax Corp,
Norcross Georgia.  Use must be consistent with the Digitrax LocoNet
licence, available at http://www.digitrax.com .

The JMRI web page is located at http://jmri.org/
